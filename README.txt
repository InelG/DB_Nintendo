Pour charger la base de donnees dans Neo4j (apres avoir cree un graphe)
- Aller dans le naviguateur (Neo4j Browser)
- options>"Enable multi statement query editor">active
- copier le contenu de **cyper_universnintendo.cql**
- le coller dans le navigateur

Les donnees etant generees en Cypher dans **db_nintendo.cql**, il n'est pas necessaire de charger le csv. Il est tout de meme present.
