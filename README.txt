Pour charger la base de donnée dans Neo4j (après avoir créé un graphe)
- Aller dans le naviguateur (Neo4j Browser)
- options>"Enable multi statement query editor">activé
- copier le contenu de **cyper_universnintendo.cql**
- le coller dans le navigateur

Les données étant générées en Cypher dans **db_nintendo.cql**, il n'est pas nécessaire de charger le csv. Il est tout de même présent.
